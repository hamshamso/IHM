# Human-Computer Interaction (HCI) - Lab Solutions 🖥️ javafx

This repository contains my personal solutions for the **IHM (Interactions Homme-Machine)** course labs. The projects are built using **Java** and **JavaFX**, focusing on GUI development and user experience principles.

---

## 📂 Lab Breakdown (TPs)

### 🔹 TP 1: Java Fundamentals Review
* **Concepts:** Quick refresher on Java syntax and logic.
* **Exercises:** Basic console and GUI interactions, such as calculating the square of an input number.

### 🔹 TP 2: Layout Management
* **Concepts:** Understanding how to organize components within a window.
* **Content:** Working with various JavaFX Layout Panes (HBox, VBox, GridPane, BorderPane) to create responsive designs.

### 🔹 TP 3: Event Handling
* **Concepts:** Making the interface interactive.
* **Content:** Implementing `EventHandler` and `Listeners` to manage user actions like button clicks, mouse movements, and keyboard inputs.

### 🔹 TP 4: Advanced JavaFX Components & Styling
* **Concepts:** Building complex and styled industrial-grade interfaces.
* **Content:** * Implementing **TableView** for data display.
    * Using **ComboBox** for selection menus.
    * Applying **CSS** (Cascading Style Sheets) to customize the look and feel of JavaFX components.

---

## 🛠 Tech Stack
* **Language:** Java
* **Framework:** JavaFX
* **Styling:** JavaFX CSS (.css)
* **Tools:** Scene Builder (optional), IntelliJ IDEA / Eclipse / VS Code.
